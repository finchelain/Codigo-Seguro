#Comando GIT local
*Clonar repositorio: Git clone <repository>
*Inicializar repositorio: git init
*Añadir cambios al stage: git add <cambios>
*Registrar cambios: git commit - "Descripción"
*Subir cambios al remoto: git push origin main
*Crear rama: git checkout -b nombrerama
*Ver rama: git branch
*Cambiar rama: git checkout rama
